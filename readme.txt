1. For database
Import the database file 'blog_manager' in the blogmanager_db folder to MySQL.

2. .env file
Change your database management system settings in 'env' file placed in blogmanagerapi folder if needed.

3. Execution of project
Right click the 'blogmanagerangular' folder, and go to cmd/bash/powershell windows of this folder. Write 'ng serve' command and enter. 
Right click 'blogmanagerapi' folder, and go to cmd/bash/powershell windows. Write 'php artisan serve' command and enter.
